\subsection{Detailed Description}
Familia printf no lpc2378 com gcc

A funcao mprintf e'parecida com a fprintf da biblioteca padrao O primeiro parametro e' o nome da funcao putchar que deve ser usada para escrever. Os codigos de formato sao parecidos com os da funcao printf padrao: Existem difierncas: so alguns formatos sao suportados. Suporta os formatos: O modificador '-\/' significa alinhamento a esquerda. O modificador '0' faz preencher o campo com zeros em vez de espacos \mbox{[}campo\mbox{]} numero de caracteres de largura total do campo Se campo for omitido assume 0; Se campo inicia com 0 preenche com zeros. \mbox{[}frac\mbox{]} Numero de caracteres da fracao do formato f; pode ser omitido. \%\mbox{[}campo\mbox{]}d ou i Escreve inteiro decimal com sinal \%\mbox{[}campo\mbox{]}u Decimal sem sinal \%\mbox{[}campo\mbox{]}x ou X Hexadecimal \%\mbox{[}campo\mbox{]}o ou q (letra o ou letra que): Escreve como octal. \%\mbox{[}campo\mbox{]}c Escreve inteiro como caractere ASCII \%\mbox{[}campo\mbox{]}s String \%\mbox{[}campo\mbox{]}.\mbox{[}frac\mbox{]}f Escreve numero tipo double com ponto fixo \%\mbox{[}campo\mbox{]}.\mbox{[}frac\mbox{]}e Escreve double em notacao cientifica \%\mbox{[}campo\mbox{]}.\mbox{[}frac\mbox{]}g Ponto fixo comutando automaticamente para notacao cientifica se o valor for maior que 1e10 ou menor que a precisao. Os transformadores de tipo h, l, L, hh ou ll sao ignorados. Caracteres de formato maiusculos e minusculos dao o mesmo resultado. Retorna o numero de caracteres escritos.

A funcao sprintf e' similar ao sprintf da biblioteca padrao. Este modulo e' independente do hardware
